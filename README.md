# Projects

A curated portfolio of health data science projects focused on equity, outcomes research, and reproducible pipelines.

---

## Overview
This repository collects modular projects that demonstrate:
- Scalable, auditable data pipelines for healthcare systems
- Equity-focused analytics and monitoring tools
- Transparent, reproducible workflows for clinical research
- Executive-ready reporting and visualization

---

## Tech Stack
- **Python**: data wrangling, modeling, pipeline engineering
- **R (tidyverse, ggplot2)**: statistical analysis and visualization
- **SQL**: schema design, relational mapping, and integration
- **Documentation**: reproducible workflows and clear reporting

---

## Project Highlights
- **Equity Drift Monitor**: Detects shifts in healthcare outcomes across patient subgroups
- **Patient Journey Graphs**: Models longitudinal care pathways for outcome analysis
- **Synthetic Rare-Disease Cohorts**: Generates simulated datasets for methodological testing
- **Geospatial Mapping**: Visualizes healthcare access disparities

---

## Methodology
- Bias mitigation and rigorous study design
- Transparent operational definitions
- Reproducible pipelines with clear audit trails
- Modular code for validation and diagnostics
